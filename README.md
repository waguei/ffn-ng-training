# ffn-ng-training
sample code for ffn angular training

## Environment 
- node & npm
- have webpack preinstall on global

After git clone then

```bash
$ npm install
$ cd ffn-ng-training
$ node server
```

Server should be launched on port 8080

### Webpack Watch mode

```bash
$ webpack -w
```
Run webpack in watch mode to build the bundle.js on-the-fly
